# 🎓 CampusIQ — University AI Assistant System

> ⚠️ This is a demo repository. Source code is private.

## 🚀 Overview
CampusIQ is an AI-powered university assistant system designed to automate and simplify academic life for students and faculty.

---

## ✨ Features

- 📊 **GPA / CGPA Calculator** — Smart academic performance tracker
- 🔮 **GPA Predictor** — Predict future GPA based on current performance and trends
- 📚 **Academic Document Search** — RAG-based document intelligence
- 🤖 **AI Study Assistant** — Personalized learning support
- ✅ **Grading System** — Shows official Grades
- 🔄 **Multi-Agent Workflow** — Agents working together seamlessly
